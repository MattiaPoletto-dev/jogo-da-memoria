# Jogo da Memória

## Objetivo
- Criar um jogo que estimulasse a lógica e o pensamento computacional
- Trabalho da faculdade

## Sobre o projeto
Este proheto foi realizado por mim, Mattia Poletto, e minha equipe Kaique Mendes e Gabriel Maier. Nesse programa, buscamos fazer um jogo da memória (conhecido com Genius), buscando fazer a lógica do programa em JavaScript, além do que esse jogo é um ótimo exercício para mente, estimulando a memória e a atenção.
